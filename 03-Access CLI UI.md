**Proxmox access through CLI or UI**


**CLI**
```
$ qm list -> list all the vms
$ pveversion
$ kvm --version
```


**Web UI**
```
-> https://IP:8006
-> Username/Password
```

**Proxmox for Mobile**

```
-> Windows Web-UI
-> https://IP:8006/?mobile=1
-> Username/Password
```

**SSH**
```
-> Putty
-> IP:22
-> root/password
```

**WinScp**
```
-> winscp
-> scp :ip: 22
-> root/password
```
