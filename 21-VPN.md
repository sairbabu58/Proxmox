### WireGuardVPN Server Configuration

```
-> VPN
  - WireGuard
  - Instances
  - Create [+]
  - Name: WireGuardVPN
  - Puplic And private Key [Click the Setting Icon]
  - Listen Port: 51825 [51820 is default]
  - Tunnel Address: 192.168.20..1/24
  - Save
  - Apply
```
